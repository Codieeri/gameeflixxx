# 🎮 GAMEFLIX

**Free Game Discovery Platform**

A Netflix-inspired web platform to explore and discover free online games with a modern UI and smooth user experience.

---

## ✨ Features

* 🎬 Cinematic intro animation
* 🎯 Hero carousel with featured games
* 🎮 Game categories (Popular, Adventure, Horror, New Releases)
* ⭐ Ratings on game cards
* 🔍 Search functionality (title, genre, developer)
* 🔐 Sign In / Sign Up UI
* 📱 Responsive design (mobile + desktop)
* 🎨 Modern dark-themed UI

## ⚙️ Setup Instructions

### 🔹 Frontend

Simply open:

```
index.html
```

---

### 🔹 Backend (Flask)

```
cd backend
pip install -r requirements.txt
python app.py
```

Runs on:

```
http://localhost:3030
```
